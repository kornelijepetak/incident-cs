# incident-cs
![Logo](http://kornelijepetak.com/incident-logo.png)

Randomization library for .NET
Inspired by the [chance.js](http://chancejs.com/) library.

## Randomization areas:
- [ ] Primitives
    - [x] Numerics
    - [x] Guid
    - [ ] Characters
    - [x] DateTime
    - [ ] Timespans
- [ ] Words and sentences
- [ ] Names and Birthdays
- [ ] Phone numbers
- [ ] Geographic data
- [ ] Colors
- [ ] Games related
- [ ] Web related

## Other features
- [ ] Consider using Mersene Twister instead of System.Random
